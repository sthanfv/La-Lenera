# **App Name**: La Leñera Digital

## Core Features:

- Visualización de Leña: Mostrar imágenes atractivas de diferentes tipos de leña y cortes disponibles para la venta.
- Cotización Rápida: Un formulario simple para que los usuarios ingresen la cantidad y el tipo de leña que necesitan para recibir una cotización rápida por WhatsApp.
- Generación de Contenido IA: Generar descripciones personalizadas y atractivas para cada tipo de leña utilizando una IA tool que analiza sus características.
- Integración con WhatsApp: Integrar la funcionalidad de WhatsApp para recibir y responder a las consultas de los clientes directamente desde la aplicación.
- Galería de Asados: Mostrar una galería de fotos de asados realizados por clientes que utilizan la leña.

## Style Guidelines:

- Color primario: Marrón óxido (#A66321), evocando la madera quemada y la tradición del asado.
- Color de fondo: Gris oscuro (#232323) para un contraste que resalte la leña y transmita calidez.
- Color de acento: Naranja fuego (#FF4500), usado para botones y elementos interactivos que inviten a la acción.
- Tipografía para encabezados y cuerpo de texto: 'Oswald' para títulos (sans-serif) y 'Inter' para cuerpo de texto (sans-serif) para una lectura clara.
- Iconos de estilo lineal en color naranja fuego (#FF4500) para representar características de la leña y servicios.
- Diseño limpio y moderno con un enfoque en imágenes de alta calidad de la leña y los asados.
- Animaciones sutiles al cargar imágenes y transiciones suaves para mejorar la experiencia del usuario.
